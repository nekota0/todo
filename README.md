# **todo_list CLI program**
you can store 255 todo into this program.
all the todos are saved at todo.txt

## **How to use**
First, run the cargo.
/
And then, you can input the command
/
/
/
to add the todo
```
add {todo_name}
```

to delete the todo
```
delete {todo_name}
```

to stop the program
```
You can input any other values to stop the program
```
